# Architecture Decision Records (ADRs)

ADRs capture the **cumulative** state of architectural decisions — not deltas. (Session 11: deltas-as-history confuses agents. If we change a decision, edit the ADR rather than appending "but actually.")

## Numbering

`docs/adrs/{NNNN}-{slug}.md`, where `NNNN` is zero-padded (`0001`, `0002`, …).
